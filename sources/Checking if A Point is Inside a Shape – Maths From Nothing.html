Checking if A Point is Inside a Shape

This page describes how to check if a point is inside a shape in any number of dimensions, including non-convex shapes. Checking if a point is inside a shape is commonly calculated on computers to determine if the user clicked a button but it can also be used to detect collisions between objects. This problem is trivial for a person to solve provided they can see the point and the boundary of the shape. If the point is enclosed within the shape it is inside the shape otherwise it is outside the shape. A computer can not solve this problem in the same way.

One method that can be used to check if a point is inside any enclosed, non-self-intersecting shape in any number of dimensions is to draw a line in any direction away from the point and count the number of times the point crosses the boundary. If the line crosses an odd number of times the point is inside the shape. Note, the line begins at the point and extends to infinity on one side of the point, it does not pass through the point.

For example, the following steps can be used to test if the point (a,b)(a,b) is inside a circle centered at the origin of a Cartesian coordinate system with variables xx and yy and a radius of rr which can be described with the following equation

x2+y2=r2x2+y2=r2

  • Define a parametric equation for a straight line beginning at the point we are checking and traveling away from it in one direction [xy]=[cd]t+[ab][xy]=[cd]t+[ab] where t>0t>0 and cc and dd are any real numbers to describe the direction such that they have a norm greater than zero.
  • Substitute the parametric equation into the equation of the circle and solve for tt to find the intersection between the parametric equation and the circle (ct+a)2+(dt+b)2=r2(ct+a)2+(dt+b)2=r2 expand c2t2+2cta+a2+d2t2+2dtb+b2=r2c2t2+2cta+a2+d2t2+2dtb+b2=r2 (c2+d2)t2+2(ac+bd)t+a2+b2−r2=0(c2+d2)t2+2(ac+bd)t+a2+b2−r2=0 solve for tt using the quadratic formula t=−2(ac+bd)±√(2(ac+bd))2−4(c2+d2)(a2+b2−r2)2(c2+d2)t=−2(ac+bd)±(2(ac+bd))2−4(c2+d2)(a2+b2−r2)2(c2+d2)
  • If tt has one positive real solution the point is inside the circle otherwise it is outside.

Another method that can be used to check if a point is inside a convex polytope is to find the convex hull of the shape’s vertexes and the point. If the point becomes part of the convex hull it is outside the polytope, otherwise, it is inside.

Checking if a Point is Inside a Polygon

For a polygon defined with coordinates (x1,y1)(x1,y1), (x2,y2)(x2,y2),…,(xn,yn)(xn,yn) where nn is the number of vertexes describing the polygon. The intersection between a line

[xy]=[cd]t+[ab][xy]=[cd]t+[ab]

from the point (a,b)(a,b) we are testing and each edge

[xy]=[xi+1−xiyi+1−yi]s+[xiyi][xy]=[xi+1−xiyi+1−yi]s+[xiyi]

of the polygon is given by

[xi+1−xiyi+1−yi]s+[xiyi]=[cd]t+[ab][xi+1−xiyi+1−yi]s+[xiyi]=[cd]t+[ab]

solving for the parameters ss and tt

[xi+1−xiyi+1−yi]s−[cd]t=[ab]−[xiyi][xi+1−xiyi+1−yi]s−[cd]t=[ab]−[xiyi]

[xi+1−xi−cyi+1−yi−d][st]=[a−xib−yi][xi+1−xi−cyi+1−yi−d][st]=[a−xib−yi]

[st]=[xi+1−xi−cyi+1−yi−d]−1[a−xib−yi][st]=[xi+1−xi−cyi+1−yi−d]−1[a−xib−yi]

This equation needs to be solved for every edge of the polygon. If t>0t>0 and 0≤s<10≤s<1 the line in the chosen direction has crossed the boundary of the polygon. Again if the number of times the line from the point crosses the boundary is odd the point is inside the polygon otherwise it is outside. The following Octave/Matlab code implements the method described above, you can use it by copying and pasting the following code into Octave online.

xy=[.2 .2
.5 .3
.7 0
1 .3
.2 .8
1 1
.1 .9];

points=rand(100,2);

if ~all(xy(end,:)==xy(1,:))
  xy(end+1,:)=xy(1,:);
end

crosstimes=zeros(rows(points),1);
for i=1:rows(points)
  a=rand(2,1);
  b=points(i,:).';
  for j=1:rows(xy)-1
    c=(xy(j+1,:)-xy(j,:)).';
    d=xy(j,:).';
    ts=[a -c]\[d-b];
    if ts(1)>0&&ts(2)>=0&&ts(2)<1
      crosstimes(i)+=1;
    end
  end
end
in=mod(crosstimes,2)==1;

figure(1);clf;
plot(points(in,1),points(in,2),'g+',points(~in,1),points(~in,2),'rx')
hold on;axis equal;
plot(xy(:,1),xy(:,2));

The above code gives an equivalent output to the inbuilt function inpolygon for points inside the polygon. The code above could be sped up by excluding points that are outside the bounding box of the polygon, choosing the direction for the line from the point to be parallel to the x-axis, and only checking lines on the boundary that have a starting and ending yy value above and below the points yy value. Also removing the loops to vectorize the code will dramatically speed up how fast it runs in Octave, or just use the inbuilt method.

Checking if a Point is Inside a Polytope

A polytope is a shape in any number of dimensions with only flat edges and faces, it is a polygon in two dimensions, and a polyhedron in three dimensions. Only a small number of modifications are required from the method described for polygons. We will still count the number of times a line in a random direction from the point that crosses the boundary, however, how we find the intersection and check it is passing through the desired section of the boundary will be approached differently.

Each face of the boundary of the polytope will be represented with the multivariable linear equation

F=N⊤X+c=0F=N⊤X+c=0

where NN is the normal of the face pointing out of the shape given by the external product equivalent to the cross product in three dimensions of the vectors describing the edges forming each simplex

N=⎡⎢ ⎢ ⎢ ⎢ ⎢ ⎢⎣(−1)0det(S{1,…,d}∖1)(−1)1det(S{1,…,d}∖2)⋮(−1)n−1det(S{1,…,d}∖d)⎤⎥ ⎥ ⎥ ⎥ ⎥ ⎥⎦N=[(−1)0det(S{1,…,d}∖1)(−1)1det(S{1,…,d}∖2)⋮(−1)n−1det(S{1,…,d}∖d)]

where

S=[X2−X1X3−X1⋯Xd−X1]S=[X2−X1X3−X1⋯Xd−X1]

is a dd by d−1d−1 matrix in which XiXi is a dd by 11 vector of vertex coordinates of the simplex describing the face and dd is the number of dimensions. The constant cc is calculated using a known point for example X1X1

N⊤X1+c=0N⊤X1+c=0

c=−N⊤X1c=−N⊤X1

If we choose the point we are testing to be AA and a line represented as a parametric equation with a parameter t>0t>0 in a random direction BB from AA to be

X=Bt+AX=Bt+A

we can substitute this into the surface equation

F=N⊤X+−N⊤X1=0F=N⊤X+−N⊤X1=0

N⊤(Bt+A)+−N⊤X1=0N⊤(Bt+A)+−N⊤X1=0

and solve for tt

N⊤Bt+N⊤A−N⊤X1=0N⊤Bt+N⊤A−N⊤X1=0

t=N⊤(X1−A)N⊤Bt=N⊤(X1−A)N⊤B

To test if the intersection of the line and face

C=BN⊤(X1−A)N⊤B+AC=BN⊤(X1−A)N⊤B+A

intersect inside the boundary of the face simplex we use a affine transform to transform the simplex on the boundary to a unit simplex

⎡⎢ ⎢ ⎢ ⎢ ⎢ ⎢ ⎢ ⎢ ⎢⎣010⋯0⋮⋱⋱⋱⋮⋮⋱⋱⋱00⋯⋯011⋯⋯⋯1⎤⎥ ⎥ ⎥ ⎥ ⎥ ⎥ ⎥ ⎥ ⎥⎦=M[X1X2⋯XdN1⋯⋯⋯1][010⋯0⋮⋱⋱⋱⋮⋮⋱⋱⋱00⋯⋯011⋯⋯⋯1]=M[X1X2⋯XdN1⋯⋯⋯1]

M=⎡⎢ ⎢ ⎢ ⎢ ⎢ ⎢ ⎢ ⎢ ⎢⎣010⋯0⋮⋱⋱⋱⋮⋮⋱⋱⋱00⋯⋯011⋯⋯⋯1⎤⎥ ⎥ ⎥ ⎥ ⎥ ⎥ ⎥ ⎥ ⎥⎦[X1X2⋯XdN1⋯⋯⋯1]−1M=[010⋯0⋮⋱⋱⋱⋮⋮⋱⋱⋱00⋯⋯011⋯⋯⋯1][X1X2⋯XdN1⋯⋯⋯1]−1

where MM is a d+1d+1 by d+1d+1 matrix describing the affine transformation of the simplex on the boundary to a simplex with unit side lengths with the vertex where the orthogonal sides meet at the origin. The intersection point CC is transformed to its position on the unit simplex with

D=MCD=MC

If all elements of DD are greater than zero and the sum of DD excluding the last element is less than one, and t>0t>0 a boundary-crossing has been found. After checking all simplexes enclosing the shape if the number of intersections is odd the point is inside the polytope otherwise it is outside.

Implimentation

The following Octave/Matlab function checks if the provided points are inside the polytope. To use it copy and paste the inpolytope and externalproduct functions into Octave online. Then copy and paste either of the three or four-dimensional examples into Octave online. Remember, face normals need to be pointing outside the shape. Also, the four-dimensional example will have issues if all faces are parallel to the coordinate axes as the transformation to the unit simplex will require the inversion of a singular matrix. This method will also work with two-dimensional data.

function in=inpolytope(points,boundvertex,faces)
% Tests if points are in the polytope defined by boundvertex and faces
%
% in=inpolytope(points,boundvertex,faces)
%
% Input:
%
% points: q by d matrix of points to test if they are in the polytope where q is the number of points and d is the number of dimensions, each row is a point.
% boundvertex: w by d matrix of vertexes defining the boundary where q is the number of points and d is the number of dimensions, each row is a vertex.
% faces: f by d matrix of vertex indexes defining the faces of the shape
%
% Output:
%
% in: q by 1 vector of true, false values. True if point in polytope.
dim=columns(boundvertex);
count=zeros(rows(points),1);
for j=1:rows(points)
a=rand(dim,1);
b=points(j,:).';
C=nan(dim,rows(faces));
for i=1:rows(faces);
N=externalproduct((boundvertex(faces(i,2:end),:)-boundvertex(faces(i,1),:))');
c=-boundvertex(faces(i,1),:)*N;
##L1=a*t+b;
##N'*X+c=0
t=(-c-N'*b)/(N'*a);
M=[zeros(dim,1) eye(dim);ones(1,dim+1)]/[boundvertex(faces(i,:),:)' N;ones(1,dim+1)];
C(:,i)=(a*t+b);
cp=(M*[C(:,i);1])(1:dim-1);
if all(cp>=0)&&sum(cp)<=1&&t>0&&~any(sum(abs(C(:,1:i-1)-C(:,i)))==0)
count(j)++;
end
end
end
in=mod(count,2)==1;
end

function w=externalproduct(A)
% Calculates the external product of n-1 column vectors of length n. A generalization of the cross product.
% w=externalproduct(A)
%
% Input:
% A: A n by n-1 matrix
%
% Output
% w: A n by 1 vector
n=rows(A);
if n-1~=columns(A)
error("externalproduct: A n by n-1 matrix is required.");
end
w=zeros(n,1);
for i=1:n
w(i)=(-1)^(i-1)*det(A((1:n)~=i,:));
end
end
